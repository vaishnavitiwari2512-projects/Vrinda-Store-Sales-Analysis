# Vrinda Store Sales Data Analysis (Excel Project)

## Project Overview

This project is based on analyzing the sales data of Vrinda Store for the year 2022 using Microsoft Excel. The goal of this project is to understand customer behavior, identify sales trends, and generate insights to improve business performance in the upcoming year.

## Objective
Vrinda Store wants to create an annual sales report for 2022 to understand its customers and increase sales in 2023.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Data Visualization (Pie Chart, Bar Chart, Combo Chart)
- Slicers (Interactive Dashboard)

## Dataset
The dataset contains:
- Order ID
- Customer details (Age, Gender)
- Product Category
- Sales Amount
- Order Status
- State & City
- Sales Channel (Amazon, Flipkart, Myntra, etc.)

## Steps Performed
1. Data Cleaning (removed duplicates, corrected values)
2. Data Processing (created Age Groups, extracted Month)
3. Created Pivot Tables
4. Built Charts (Bar, Pie, Combo)
5. Designed Interactive Dashboard using Slicers
6. Generated Insights

## Business Questions (for which Pivot Tables were created)

1. Compare the **Sales and Orders** using a single chart  
2. Which **month has highest sales and orders**?  
3. Who purchased more — **Men or Women**?  
4. What are the **different order statuses**?  
5. Which are the **Top 5 states contributing to sales**?  
6. What is the **relationship between Age and Gender** based on orders?  
7. Which **sales channel contributes the most**?  

## Key Insights

- **March** has the highest sales and orders  
- **Women customers contribute more (~60–65%)**  
- Most orders are **successfully delivered**  
- Top states: **Maharashtra, Karnataka, Uttar Pradesh, Telangana, Tamil Nadu**  
- **Adult age group (30–49 years)** contributes the most  
- **Amazon, Flipkart, Myntra** are top sales channels  
- Sales are not consistent across months (fluctuation observed)


## Conclusion

To improve sales, Vrinda Store should:
- Target **women customers (30–49 age group)**
- Focus on **top-performing states**
- Provide **offers and discounts on Amazon, Flipkart, Myntra**
- Run **marketing campaigns during high-performing months**
